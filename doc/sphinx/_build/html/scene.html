scene – Support for 3D Scenes¶

class scene.Frame3D(scale=1.0, translation=None, rotation=None)¶

A frame of reference for applying a scale, translation, and rotation to a collection of Thing3D objects. It can also have descendant Frame3D objects that contain their own Thing3D objects.

Parameters:
  • scale (One of math3d.scalar_types) – A scaling factor.
  • translation (math3d.Vector or any iterable of any three math3d.scalar_types) – Specifies XYZ translation.
  • rotation (math3d.Matrix or any iterable of any three math3d.scalar_types) – Specifies a rotation around each of the X, Y, and Z axises. If given as discrete values (i.e. not a math3d.Matrix) they are interpreted as degrees.
Raises:

TypeError if the constraints on parameters are not met.

Each of the above three parameters becomes an instance attribute of the resulting Frame3D object that you can manipulate at runtime to affect subsequent calls to yieldTransformed().

s¶
(Instance attribute.) Scale factor applied by yieldTransformed().
T¶
(Instance attribute.) math3d.Vector Translation applied by yieldTransformed().
RM¶
(Instance attribute.) math3d.Matrix Rotation applied by yieldTransformed().
things¶
(Instance attribute.) A list of Thing3D objects. Append Thing3D objects to this list to have them yielded by yieldTransformed().
subframes¶
(Instance attribute.) A list of Frame3D objects. Append Frame3D objects to this list to have their Thing3D objects yielded by yieldTransformed().
transform(vector)¶

Return a Vector that is the result of appling the scale, translation, and rotation defined by this Frame3D to the given input Vector.

Parameter:vector (math3d.Vector) – A Vector representing a 3D point.
Return type:math3d.Vector
yieldTransformed()¶

Yield each Thing3D in this Frame3D and its descendant Frame3D objects, modified by its scale, translation, and rotation. This is called recursively on descendant Frame3D objects.

Return type:generator of two-tuples: (Thing3D, transformed math3d.Vector)
